## Overview
On-TASK is a website created to simplify and encourage adults, teens, schools, groups, and businesses to do service with certain organizations. The website includes a profile system with an experience tracker. Service providers can upload training videos and for users a service provider chatbot is included. This creates easy access and incentives to do more service. It can be used by organizations to encourage and manage.
